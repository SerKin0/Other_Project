## Console Snake
Консольная версия знаменитой игры Змейки.
Имеется:
- Движение кнопками <b>wasd</b>;
- Окончание либо ударом об стену или хвост, либо нажатием кнопки <b>x</b>;
- Подсчет очов за каждое съеденное яблоко;
